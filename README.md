## How to Install

Open the terminal and type following commands!!!

* ```apt update```
* ```apt upgrade```
* ```pkg install git```
* ```termux-setup-storage```
* ```pkg install pip```
* ```pkg install python```
* ```git clone https://github.com/rixon-cochi/telegram_bitcon_bot```
* ```cd telegram_bitcoin_bot```
* ```unzip telethon.zip```
* ```unzip session.zip```
* ```pip install -r requirements.txt```
* ```python bot.py```
